# @rui/styles

[Emotion](https://emotion.sh/) based css-in-js for Robin-UI

## Installation

```sh
# yarn
$ yarn add @rui/styles @emotion/react@^11 @emotion/styled@^11

# npm
$ npm i @rui/styles @emotion/react@^11 @emotion/styled@^11
```
