# @robin-ui/styles

[Emotion](https://emotion.sh/) based css-in-js for Robin UI

## Installation

```sh
# npm
$ npm i @robin-ui/styles @emotion/react@^11 @emotion/styled@^11

# yarn
$ yarn add @robin-ui/styles @emotion/react@^11 @emotion/styled@^11
```
