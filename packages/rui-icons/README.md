# @rui/icons

A React icon library made with [Feather](https://feathericons.com/) (4.29.0)

## Installation

```sh
# yarn
$ yarn add @rui/icons @emotion/react@^11 @emotion/styled@^11

# npm
$ npm i @rui/icons @emotion/react@^11 @emotion/styled@^11
```
