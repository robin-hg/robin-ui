# @robin-ui/icons

A React icon library made with [Feather](https://feathericons.com/) (4.29.0)

## Installation

```sh
# npm
$ npm i @robin-ui/icons @emotion/react@^11 @emotion/styled@^11

# yarn
$ yarn add @robin-ui/icons @emotion/react@^11 @emotion/styled@^11
```
