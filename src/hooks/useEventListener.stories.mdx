# useEventListener

A shorthand custom `useEvent` hook that automatically adds/removes an event listener on mount/unmount.

`const useEventListener: <K extends keyof GlobalEventHandlersEventMap>(eventName: K, listener: GlobalEventHandlersEventMap[K]) => void, dependencies?: any[]) => void`
