# useOnClickOutside

Adds `mousedown`/`touchstart` listeners that fires `action` if the `target` of the event is not in `exclude`.

`const useOnClickOutside: (exclude: HTMLElement | (HTMLElement | null)[] | null, action: (event: Event | MouseEvent | TouchEvent) => void) => void`
